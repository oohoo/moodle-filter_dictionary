--------------------------------------------------------------------------------
---------------------------- OOHOO - Popup Dictionary  -------------------------
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
Description
--------------------------------------------------------------------------------

OOHOO Pop-up Dictionary is a Moodle package that gives the students the 
possibility to double click on any word within a Moodle HTML resource and 
retrieve its definition from an online dictionary chosen by the teacher.


--------------------------------------------------------------------------------
Prerequisites
--------------------------------------------------------------------------------

Moodle 2.4.X & Moodle 2.5.X
Download and install oohoo/moodle-block_dictionary

--------------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------------

 1. Rename the folder to 'dictionary'
 2. Copy the folder dictionary to moodle/filter
 3. Install oohoo/moodle-block_dictionary from GitHub with README instructions
 3. Install the plugins (filter and block)
 4. Go to the plugin settings to add custom dictionaries
 5. Go to the filters settings to activate the dictionary filter


--------------------------------------------------------------------------------
